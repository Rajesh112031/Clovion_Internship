About the Book Catalog

The Book Catalog is a dynamic platform designed to provide book enthusiasts with a comprehensive resource for exploring literature. Built with HTML, CSS, and JavaScript, it integrates the Google Books API to fetch detailed information, including titles, authors, genres, publication years, descriptions, ratings, publishers, ISBNs, and page counts, displayed in an interactive modal.

Features include a responsive grid layout with larger book cards, a genre filter, and a search function, all optimized for seamless use across mobile, tablet, and desktop devices. The site showcases a modern design inspired by educational templates, with smooth scrolling, full-length banners, and a collapsible card view for an enhanced user experience.

Future plans include adding sorting options, external purchase links, and user reviews to further enrich the platform.
